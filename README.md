# Job Portal Management System

A desktop-based Job Portal Management System developed using C# WinForms and SQLite database.

## Features

### User Features
- User Registration & Login
- Browse Jobs
- Search Jobs
- Apply for Jobs
- View Applied Jobs
- Update User Profile

### Admin Features
- Manage Users
- Change User Roles
- Manage Jobs
- Manage Applications
- Update Application Status

## Technologies Used

- C#
- .NET Framework
- WinForms
- SQLite
- ADO.NET
- Git & GitHub
